# Welcome to Minions Comms

We're excited to introduce **Minions Comms** — Messages, threads, and notifications across all communication channels.

## What's Included

- **TypeScript SDK**: `npm install @minions-comms/sdk`
- **Python SDK**: `pip install minions-comms`
- **CLI**: `npm install -g @minions-comms/cli`

## Getting Started

Check out the [documentation](https://comms.minions.help) for a full guide.
