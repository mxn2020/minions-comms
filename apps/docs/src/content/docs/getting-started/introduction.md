# Minions Comms

Messages, threads, and notifications across all communication channels

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-comms/sdk`) — Core library for Node.js
- **Python SDK** (`minions-comms`) — Python bindings
- **CLI Tool** (`@minions-comms/cli`) — Command-line interface
- **Web App** — Interactive playground at [comms.minions.wtf](https://comms.minions.wtf)
