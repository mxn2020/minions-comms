---
title: Installation
description: Install Minions Comms packages
---

## TypeScript / Node.js

```bash
npm install @minions-comms/sdk minions-sdk
```

## Python

```bash
pip install minions-comms
```

## CLI

```bash
npm install -g @minions-comms/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
